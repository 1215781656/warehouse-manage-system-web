# 色布出入库管理系统

一个专业的纺织品库存管理解决方案，基于 Vue3 + TypeScript + Element Plus 构建。

## 功能特性

### 🏠 仪表盘

- 关键指标概览（今日出入库量、库存总量、应收应付总额）
- 库存预警提醒
- 快捷操作入口

### 📦 库存管理

- **入库管理**：色布信息录入、供应商管理、入库单生成
- **出库管理**：客户管理、出库单生成、发货单打印、库存校验
- **库存查询**：实时库存查询、库存预警、库存统计

### 💰 财务管理

- **应收管理**：应收账款跟踪、收款状态管理、账龄分析
- **应付管理**：应付账款跟踪、付款计划管理、付款登记

### 📊 报表中心

- **库存报表**：库存结构分析、库存趋势图、库存周转率分析
- **财务报表**：应收账龄分析、应付账龄分析、收支趋势图
- **业务分析**：出入库趋势、客户交易排行、供应商交易排行

### 🔧 系统功能

- 响应式设计，支持移动端
- 数据导出功能（Excel）
- 权限管理（基于角色的访问控制）
- 操作日志记录

## 核心业务流程

### 入库流程

1. 录入供应商信息和色布基础信息
2. 系统自动生成色布 ID（CB\_年份+月份+流水号）
3. 录入入库数量、重量、单价
4. 系统自动计算金额（单价 × 重量）
5. 保存入库记录，更新库存数据

### 出库流程

1. 选择要出库的色布（按色号或品名规格搜索）
2. 系统校验库存余量，不足时给出警告
3. 录入客户信息、出库数量、单价
4. 生成编号（HD\_年份+月份+客户代码+流水号）
5. 扣减库存，生成应收记录

### 库存管理

- 库存匹数 = 累计入库匹数 - 累计出库匹数
- 库存重量 = 累计入库重量 - 累计出库重量
- 库存预警：低于安全库存时标红显示
